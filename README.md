# CapCut Sanitizer

Script sederhana untuk membersihkan metadata tertentu pada video hasil template CapCut sebelum diunggah.

## Install

```bash
pkg update && pkg upgrade -y
pkg install git python -y

git clone https://github.com/Durko01/Capcuts-Sanitizer-.git
cd Capcuts-Sanitizer-

pip install -r requirements.txt

termux-setup-storage
python main.py
```

## Fungsi

- Membersihkan metadata pada video hasil template CapCut.
- Membantu mengurangi kemungkinan video menampilkan tautan template CapCut saat diunggah ke TikTok.
- Mudah digunakan langsung dari Termux.

## Catatan

- Berikan izin penyimpanan ke Termux sebelum menjalankan script.
- Tidak ada jaminan hasil akan selalu sama karena mekanisme TikTok dan CapCut dapat berubah sewaktu-waktu.
